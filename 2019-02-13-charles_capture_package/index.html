Charles 抓包（http/https）

Written by Colynn.Liu with ♥ on February 13, 2019 - last modified February 13, 2019
tools
工具

运行环境

  • Mac os 10.13.6
  • Ios 12.1.2(16C101)

下载安装

  • 官网下载
  • 注: 破解版麻请自行搜索下载

配置

  1. Proxy -> Proxy Settings (根据需要调整配置，保存)；

注: 如果仅需要抓取http包，请略过后续步骤。

  1. Proxy -> SSL Proxying Settings

  1. 电脑端安装证书 Help -> SSL Proxying -> Install Charles Root Certificate，并将证书标记为始终信任

  1. 手机端安装证书, 用Safari浏览器打开： chls.pro/ssl ；提示下载， 然后安装 注: IOS 10.0之后需要在 设置 -> 通用 -> 关于本机 -> 证书信息设置， 信任该证书就可以了。

  2. 获取代理信息： 通过如下方式获取代理信息： Help -> SSL Proxying -> Install Charles Root Certificate on a Mobile Device or Remote Browser， 显示如下图 代理信息为： 172.16.17.35:8888

  1. 手机端配置代理： 设置 -> 无线局域网 -> 网络id -> 配置代理 -> 手动, 如下图
